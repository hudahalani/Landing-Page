FPGA Programmer - Brain Blitz Game

ECE241 - Digital Logic Design

Project Overview

Developed Brain Blitz, an interactive FPGA-based quiz game on the DE1-SoC board using Verilog. Features mental math, logic, and pattern recognition questions with VGA display and PS2 keyboard input. Players race against a countdown timer with instant game-over on wrong answers.

Key Features

  • 13-State FSM: Controls homepage, questions, win/loss states
  • VGA Display: 640x480 resolution with dynamic screen multiplexing
  • PS2 Keyboard: Real-time input processing for user responses
  • Hardware Integration: LED score display and HEX countdown timer
  • Multiple Question Types: Mental math, logic puzzles, pattern recognition

Technical Implementation

  • Hardware: Intel DE1-SoC FPGA board
  • Language: Verilog HDL
  • Display: VGA interface with screen memory management
  • Input: PS2 keyboard protocol implementation
  • Timing: Real-time countdown and game state management

Skills Developed

Gained expertise in digital logic design, Verilog programming, finite state machines, and hardware-software integration. Learned real-time system design and FPGA development using Quartus Prime and ModelSim.
