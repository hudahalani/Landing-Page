FPGA Programmer - Brain Blitz Game

ECE241 - Digital Logic Design

Project Overview

As part of ECE241, I developed Brain Blitz, an interactive, FPGA-based quiz game built on the DE1-SoC board using Verilog. Designed to test mental math, logic, and pattern recognition, the game displays multiple-choice questions on a VGA screen and receives input from a PS2 keyboard. Players race against a countdown timer, with the game ending instantly upon a wrong answer to keep the pace intense and engaging.

Technical Implementation

Hardware Architecture

Brain Blitz leverages the DE1-SoC FPGA board's capabilities for real-time game processing:

  • DE1-SoC Board: Intel Cyclone V SoC with ARM Cortex-A9 processor
  • VGA Display: 640x480 resolution for crisp question presentation
  • PS2 Keyboard Interface: Real-time input processing for user responses
  • LED Indicators: Visual feedback for game state and scoring
  • HEX Displays: Countdown timer and score visualization

Finite State Machine Design

The game is driven by a sophisticated 13-state Finite State Machine (FSM) that controls all aspects of gameplay:

  • State Management: Homepage, question display, input processing, win/loss states
  • Dynamic Transitions: Smooth state changes based on user input and timer events
  • Error Handling: Robust state recovery and system stability
  • Performance Optimization: Minimal clock cycles for responsive gameplay

Display System

Advanced VGA display management with dynamic content rendering:

  • Screen Multiplexer: Dynamic VGA memory selection for real-time content updates
  • Question Storage: Multiple screens stored in memory with dynamic selection
  • Visual Feedback: Real-time score and timer display on hardware indicators
  • Responsive Design: Optimized rendering for smooth visual transitions

Input Processing

Seamless integration of multiple input sources for intuitive gameplay:

  • PS2 Keyboard Interface: Real-time key detection and processing
  • Input Validation: Robust error checking and response handling
  • Multi-format Support: Handles various question types and input methods
  • Low Latency: Minimal delay between input and system response

Game Features

  • Multiple Question Types: Mental math, logic puzzles, and pattern recognition
  • Countdown Timer: Pressure-based gameplay with visual countdown display
  • Instant Feedback: Immediate response to correct/incorrect answers
  • Score Tracking: Real-time score display using LED indicators
  • Progressive Difficulty: Increasingly challenging questions as game progresses
  • Hardware Integration: Full utilization of DE1-SoC board capabilities

Technical Skills Developed

  • Verilog Programming: Hardware description language for FPGA design
  • Digital Logic Design: Complex circuit design and optimization
  • State Machine Design: Finite state machine implementation and debugging
  • Hardware-Software Integration: Seamless interaction between FPGA and peripherals
  • Real-time Systems: Timing-critical applications with strict performance requirements
  • Debugging Techniques: Hardware debugging using oscilloscopes and logic analyzers

Technologies Used

  • Hardware Description Language: Verilog HDL
  • Development Board: Intel DE1-SoC FPGA
  • Display Interface: VGA (640x480 resolution)
  • Input Interface: PS2 keyboard protocol
  • Development Tools: Quartus Prime, ModelSim
  • Hardware Peripherals: LEDs, HEX displays, switches

Learning Outcomes

This project provided hands-on experience with digital logic design and FPGA programming, strengthening my understanding of:

  • Digital Circuit Design: Complex logic circuits and optimization techniques
  • Hardware Timing: Clock domain management and timing constraints
  • System Integration: Multiple hardware components working together
  • Real-time Programming: Time-critical applications and performance optimization
  • Hardware Debugging: Troubleshooting complex digital systems

Impact

Brain Blitz successfully demonstrates the capabilities of FPGA-based gaming systems, showcasing how hardware description languages can create engaging, real-time interactive experiences. The project highlights the importance of efficient state machine design and hardware-software integration in embedded systems development.
