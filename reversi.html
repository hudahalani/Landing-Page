Reversi Game in C

University of Toronto Programming Course

Project Overview

As part of a coding course at the University of Toronto, I was tasked with programming the classic 'Reversi' game and competing against other students. This project challenged me to implement complex game logic, develop an intelligent AI opponent, and create a user-friendly terminal interface using the C programming language.

Game Implementation

Core Game Logic

Developed a complete implementation of the Reversi game with comprehensive features:

  • Game Board Management: 8x8 grid implementation with proper state tracking
  • Move Validation: Comprehensive rules checking for valid moves
  • Piece Capturing: Accurate implementation of the flipping mechanism
  • Game State Tracking: Real-time monitoring of board state and player turns
  • Win Condition Detection: Automatic game termination when board is full

User Interface

Created an intuitive terminal-based interface for seamless gameplay:

  • Visual Board Display: Clear representation of the game board using ASCII characters
  • Input Validation: Robust error handling for user input
  • Game Status Updates: Real-time display of current player, valid moves, and score
  • Move History: Tracking and display of previous moves
  • Help System: Built-in instructions and game rules

Advanced Features

AI Opponent

Developed an intelligent computer opponent with strategic gameplay:

  • Move Selection Algorithm: AI that finds and executes the best available moves
  • Strategic Decision Making: Implementation of game theory principles
  • Difficulty Levels: Adjustable AI intelligence for different skill levels
  • Performance Optimization: Efficient algorithm design for fast move calculation

Game Modes

Implemented multiple gameplay options to enhance user experience:

  • Player vs Player: Two human players competing on the same terminal
  • Player vs Computer: Human player against the AI opponent
  • Computer vs Computer: AI-only games for testing and demonstration
  • Tournament Mode: Support for competitive play against other students

Technical Implementation

Data Structures

Efficient data organization for optimal performance:

  • 2D Array Implementation: Efficient board representation using arrays
  • Move Tracking: Data structures for storing and analyzing moves
  • State Management: Comprehensive game state tracking

Algorithm Design

Sophisticated algorithms for game logic and AI:

  • Move Generation: Algorithm to find all valid moves for a player
  • Move Evaluation: Scoring system for determining move quality
  • Minimax Algorithm: Implementation of game tree search for AI decisions
  • Alpha-Beta Pruning: Optimization technique for faster AI computation
