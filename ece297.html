SafeRoute: A Safety-First Navigation System

ECE297 - Software Design and Communication

Project Overview

As part of a three-person team, I co-developed SafeRoute, a safety-first navigation platform aimed at empowering safer campus movementâ€”especially for women. Built using C++ and a custom graphics engine, SafeRoute integrates OpenStreetMap data with advanced pathfinding algorithms to deliver fast and intelligent routing with a focus on user safety and accessibility.

Technical Implementation

Core Architecture

SafeRoute is built using C++ with a custom graphics engine, providing a robust foundation for real-time navigation:

  • Custom Graphics Engine: Developed from scratch to handle map rendering and user interface
  • OpenStreetMap Integration: Real-time data parsing and route calculation
  • Advanced Pathfinding: Implementation of Dijkstra's, A*, and Simulated Annealing algorithms
  • Performance Optimization: Achieved pan/zoom speeds under 100ms for smooth user experience

Safety Features

Safety is the core focus of SafeRoute, with multiple features designed to protect users:

  • POI Safety Ratings: Integration with Google Places API to provide safety ratings for points of interest
  • Emergency Location Sharing: SOS button with rapid location sharing via POST requests using libcurl
  • Trusted Contact System: Ability to alert campus security, trusted contacts, or emergency services
  • Intersection-Based Location: Click-to-select nearest intersection for precise emergency location

User Interface & Accessibility

Designed with accessibility and user experience in mind:

  • WCAG Compliance: Verified using Pa11y for accessibility standards
  • Dark Mode Support: Enhanced visibility and reduced eye strain
  • Dynamic Distance Scale: Real-time distance calculations and display
  • Collapsible Sidebar: Step-by-step navigation instructions
  • Color-Coded Subway Lines: Clear visual representation of transit options

Emergency Response System

Comprehensive emergency features for rapid response in critical situations:

  • SOS Button: One-click emergency alert system
  • Location Precision: Intersection-based location sharing for accurate emergency response
  • Multiple Alert Channels: Campus security, trusted contacts, and emergency services integration
  • Rapid Response: Optimized POST requests for immediate alert transmission

Key Features

  • Real-time Navigation: Live route updates and traffic-aware pathfinding
  • Safety-First Routing: Algorithms optimized for well-lit, populated routes
  • Emergency Integration: Seamless connection with campus security systems
  • Accessibility Compliance: Full WCAG 2.1 AA compliance for inclusive design
  • Performance Optimized: Sub-100ms response times for smooth interaction
  • Multi-Platform Ready: Architecture designed for future mobile expansion

Technologies Used

  • Programming Language: C++
  • Graphics Engine: Custom-built rendering system
  • Mapping Data: OpenStreetMap API
  • Safety Data: Google Places API
  • Network Communication: libcurl for HTTP requests
  • Version Control: Git and GitHub
  • Accessibility Testing: Pa11y

Team Collaboration

This project strengthened my skills in collaborative software development, including:

  • Git Workflow: Efficient branching, merging, and code review processes
  • Team Coordination: Three-person development with clear role distribution
  • Code Quality: Consistent coding standards and documentation
  • Project Management: Agile development with regular milestones and reviews

Impact & Results

SafeRoute achieved exceptional user experience metrics with a System Usability Score of 91, demonstrating the effectiveness of our safety-first approach. The platform successfully balances performance, accessibility, and security features, making it a valuable tool for campus navigation.

Future Development

We're excited to explore future expansion into mobile platforms with enhanced features including crime heat maps, real-time emergency support, and integration with campus security systems for comprehensive safety coverage.
